============ para instalar o virtual Env =============


    python -m venv  "nome do ambiente virtual"


============ para ativar o virtual Env ================


    No Windows, execute: .\env\Scripts\activate.bat

    No Unix ou MacOS, execute: ./env/bin/activate

    quando utilizar o shell bash : activate.csh ou activate.fish


============= instalar os pacotes no virtual env ==========


    pip install "nome do pacote de deseja instalar"


========= pode realizar update no pacote do virtual env ===

    pip install --upgrade


==== para lista todos os pacotes instalado no virtual env ==


    pip list 

    ex: (tutorial-env) $ pip list
            novas (3.1.1.3)
            numpy (1.9.2)
            pip (7.0.3)
            requests (2.7.0)
            setuptools (16.0)


==== para exibir informações sobre um pacote específico ====


    pip show "nome do pacote"

==== para criar uma lista com os pacotes instalado no virtual env ===


    pip freeze > requirements.txt

    > -> informa
    
    requirements -> serar o nome da lista 

    .txt -> tipo de arquivo que vai ser salvo 

 ==== pode ser utilizado para instalr os pacote de uma aplição =======

    
    pip install -r requirements.txt

    -r -> indica o arquivo com os pacotes para ser instalado na maquina de uma vez só
